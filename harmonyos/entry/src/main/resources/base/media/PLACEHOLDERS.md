# HarmonyOS 媒体资源占位符

这个目录包含 HarmonyOS 应用需要的资源文件占位符。

## 真实开发时需要

### 应用图标
- app_icon.png (不同尺寸：128x128, 192x192, 256x256 等)
- 或者使用 SVG 格式

### 启动图标
- start_icon.png

### 模块图标
- module_icon.png

## 说明

这些占位符是为了项目结构完整性。在真实的开发环境中，
你需要使用 DevEco Studio 的资源管理器来导入真实的图片资源，
或使用 HarmonyOS 的 Image 组件从网络加载。

注意：鸿蒙系统建议使用以下尺寸的图标：
- 手机应用图标：128x128px
- 平板应用图标：192x192px
- 圆形图标：256x256px
